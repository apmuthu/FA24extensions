How To Use This Report To Print And Update Checks
  1. Select the starting bank payment to be printed. The first one in the displayed list is the next bank payment after the last one that was printed. This is determined by looking for the pattern [] in the check "Memo:".
  2. Do a dry run first to review the PDF.
  3. Once the PDF is correct, load the checks and print.
  4. If the checks print okay, set Update Check Numbers to "Yes" and rerun the report. This updates the checks by adding the check number in square brackets in the "Memo:" field.
  5. Use the Ending Check Number field to limit the number of checks to print or update. This is useful if some checks failed to print properly due to printer problems.
